# InteliJ 단축키
- Alt + Enter = Show context actions / static import
- Ctrl Ctrl = run anything
- Ctrl + Shift + Enter = 구문완성: 세미콜론, 반복, 조건문 등
- Ctrl + Shift + Alt + T = 리팩토링
- Shift + F6 = Rename
- Ctrl + Alt + V = 변수 추출
- Shift + Ctrl + T = 테스트 작성
- Ctrl + Alt + N = Inline Variable
