# 바킹독 실전 알고리즘 강좌

- 코테를 위한 능력
  - 배경지식: 알고리즘, 자료구조, 테크닉
  - 문재해결능력: 배경지식의 활용력
  - 구현력: 풀이를 코드로 옮겨낼 수 있는 능력



## 기초 코드 작성 요령

- **시간복잡도(Time Complexity)**
- **빅오표기법(Big-O Notation)**
- **공간복잡도(Space Complexity)**
  - 입력의 크기와 문제를 해결하는데 필요한 공간의 상관관계
  - 코테에서는 중요성이 덜함
  - 단, 다음을 기억
    - 메모리 512MB = 1.2억개의 int
- **자료형**
  - 자료형의 크기와 오버플로우 주의
  - 실수표기법
    - 오차 발생 가능
    - 자료 변환 주의
    - 등호 사용 비교하지 말자
      - `abs(a-b) < 1e-12`와 같은 식의 차이 비교
